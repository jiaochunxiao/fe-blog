### Jest 匹配器

每次要测试一个值时都会使用expect函数。 你很少会自己调用expect。 相反，您将使用expect及 "matcher" 函数来断言某个值。

+ .toBe(value)
+ .toBeNull()
+ .toBeDefined()
+ .toBeUndefined()
+ .toBeTruthy()
+ .toBeFalsy()
+ .toBeNaN()
+ .toBeGreaterThan(number | bigInt)
+ .not
+ ...
